Proxy install

ATTENTION. Please don`t use your main account for getting profiles from ingress.

Download first this certificate for proxy.

1. Close ingress app and Go to Settings > Privacy > Install from storage (Install sertificates from storage)

2. Choose certificate that you downloaded. It may take to put the lock screen pin.

3. Insert any name of cert and choose "VPN and app" (not wifi) in credentials use.

After you will get message that your network may be monitored.

4. Go to Wifi menu, select your connection and modify proxy settings.

5. Go to and add account name. The service will not work if you add incorrect name.

6. Now you can start your ingress app. You can disable proxy when inventory will be downloaded.

Refresh this page. If you see factions color on account name then all work properly.

7. Now you can see profiles without scanner and hide achievements of account. And you can see inventory of account that was connected to service.
